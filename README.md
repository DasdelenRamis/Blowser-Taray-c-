## 📌 Özellikler

### 🎨 Arayüz
- **Modern Tasarım** - Qt tabanlı şık ve sade arayüz
- **Özel Sekme Şeridi** - Üstte düzenlenebilir sekmeler
- **Yer İmleri Çubuğu** - Hızlı erişim araç çubuğu
- **Koyu Tema** - Göz yormayan gece modu
- **Güvenlik Göstergesi** - HTTPS/HTTP durum ikonları

### 🚀 Fonksiyonlar
- ✅ **Sekmeli Gezinti** - Çoklu sekme desteği, sürükle-bırak ile sıralama
- 🕶️ **Özel Gezinti** - İz bırakmayan gizli mod
- ⭐ **Yer İmleri** - Ekle, sil, ara, düzenle
- 📜 **Geçmiş** - Ziyaret sayacı, arama ve temizleme
- 📥 **İndirme Yöneticisi** - İlerleme çubuğu ve durum takibi
- 🔍 **Sayfa İçi Arama** - Kolay bulma aracı
- 🔎 **Yakınlaştırma** - 3 katına kadar zoom
- 🖨️ **Yazdırma** - PDF ve kağıt çıktısı

### ⌨️ Kısayollar
| Kısayol | İşlev |
|---------|-------|
| `Ctrl+T` | Yeni sekme |
| `Ctrl+W` | Sekmeyi kapat |
| `Ctrl+Shift+P` | Özel sekme |
| `Ctrl+D` | Yer imi ekle/çıkar |
| `Ctrl+H` | Geçmişi aç |
| `Ctrl+Shift+B` | Yer imlerini aç |
| `Ctrl+F` | Sayfada bul |
| `Ctrl+L` | Adres çubuğuna odaklan |
| `F5` | Sayfayı yenile |
| `Alt+Left/Right` | Geri/İleri |
| `Ctrl++/-/0` | Yakınlaştırma |
| `Ctrl+Tab` | Sonraki sekme |
| `Ctrl+Shift+Tab` | Önceki sekme |
